About AuthTask

AuthTask is designed to demonstrate a polished Django authentication flow with modern, custom styles and smooth button motion.

Built with a clean component-based UI, this sample app separates shared page structure from individual content pages using Django templates.

The navigation bar, typography, and button interactions are all managed with internal CSS, so the interface remains lightweight and professional.

  • Login form
  • Signup form
  • About / project summary page
